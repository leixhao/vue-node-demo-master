### 本地测试环境：  
#### 第一步：找到config文件里面的mysql.json文件，把里面的数据库配置改成自己的信息.  
#### 第二步：npm install 把项目依赖安装.  
#### 第三步：npm run sqlInit 初始化数据表.  
#### 第四步：npm run dev 打开一个终端，运行项目.  
#### 第五步：npm run api 另外打开一个终端，运行node API.
